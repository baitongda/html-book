# &lt;h1&gt;..&lt;h6&gt;

&lt;h1&gt; 到 &lt;h6&gt; 是在 HTML 里可以使用的六个级别的标题，&lt;h1&gt; 标题级别最大，&lt;h6&gt; 标题级别最小。使用这些标题标签你可以划分内容的关系，比如一篇故事内容，一级标题可能是故事的名字，二级标题是故事章节的名字，每个章节下面的不同的情节的名字，可以使用三级标题。

## 用法

```
<h1>标题一</h1>
<h2>标题二</h2>
<h3>标题三</h3>
<h4>标题四</h4>
<h5>标题五</h5>
<h6>标题六</h6>
```
